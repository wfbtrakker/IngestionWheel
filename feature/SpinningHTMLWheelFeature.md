# Feature: Spinning Wheel

## Overview
Web application that displays a circular wheel that spins and eventually stops.  The wheel should be made up of pie slices that represent user names loaded from a list.  When it spins it should stop on the name of one of the users.

## Problem Statement
Want to randomly select a user by spinning a wheel.

## Goals
- Select a user randomly from the wheel as it spins and stops.
- Colorful and visibly pleasing wheel.

## User Stories
As a user, I want to spin a wheel, so that it spins and randomly stops on a users name.

**Acceptance Criteria:**

Users are selected randomly.

## Technical Approach

### Architecture
Only front end scripting to be used like HTML, CSS, Javascript, etc.
